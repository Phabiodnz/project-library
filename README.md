# Dynamic Library Displayer

A small web app to manage your personal book collection. Add, view, update, and remove books dynamically.

## Features

- Add new books with title, author, pages, and read status
- Display books as interactive cards
- Toggle read status or remove books

## Usage

1. Open `index.html` in your browser
2. Click **"ADD BOOK"** to add a new book
3. Use **"REMOVE"** or **"CHANGE STATUS"** buttons on each book card

## Tech

- HTML, CSS, JavaScript (Vanilla ES6+)
